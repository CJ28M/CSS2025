Media Queries

The "@media" CSS at rule can be used to apply of a style sheet based on the result of one or more media queries. With it, you specify a media query and a block of CSS to apply to the document if and only if the media query matches the device on which the content is being used.

Media types

A "media-type" describes the generak category of a device. Except when using the "only" logical operator, the media type is optional and the all type is implied.

all

Suitable for all devices.

print

Intended for paged material and documents viewed on ascreen in print preview mode.

screen

Intended primarily for screens.
