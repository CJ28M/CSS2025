Positioning

  • Static- Follows the normal flow
  • Relative- Gets positioned relatively to its normal position
  • Absolute- Gets absolutely positioned to nearest positioned parent or the html element
  • Fixed- Gets fixed positioned to the viewport
  • Sticky- Acts like a relatively positioned until it reaches the constraint value
    • Then starts acting like a fixed element until it reaches the end of it's parent block.
    • Gets attached to the nearest scrolling mechanism
    • Then keeps acting like a relatively positioned element
