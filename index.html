How to split screen your VS Code

  1. set your Html file inluding the boilerplate
  2. run: Go Live
  3. copy the URL path
  4. Shift+Ctrl+P (in VS Code)
  5. find "Simple Browser:Show"
  6. Paste Path URL and enter
  7. right click on the tab of "Simple Browser"
  8. Click on split right
  9. Drag the "Simple Browser" to the right box

* Note: You can not inspect the view port!
